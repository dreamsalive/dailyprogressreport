    RUle of 3 All should be defined if a user implements any of them Destructor Copy constructor Copy assignment operator This will be due to allocation of some resource in a constructor Destructor will free the resource Copy constructor will perform a deep copy Copy assignment operation will also perform a deep copy Not implementing user defined operations can lead to memory leak or shallow copy


    Rule of 5 All should be defined if a user implements any of the following Destructor Copy constructor Copy assignment operator Move constructor Move assignment operator
